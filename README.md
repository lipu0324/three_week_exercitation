# three_week_exercitation

* 前端使用vue框架，后端使用springboot框架，数据库使用mysql
* 项目原型使用墨刀设计，原型地址[墨刀](https://modao.cc/proto/NFNYEifrwj5fk7vjCvwgM/sharing?view_mode=read_only)
